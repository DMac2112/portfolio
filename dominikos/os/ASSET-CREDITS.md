> It is not affiliated with, endorsed by, or connected to Microsoft Corporation.

Policy (DOMINIKOS-PLAN §6): if an asset can be traced to a Windows install or an official
Microsoft download, it does not exist in this repo. Everything visual is an original redraw
that *evokes* the era. `npm run gate` enforces this on the repo + built output.
